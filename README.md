# Canopy Portfolio

暗黑金赛博极简风格的个人品牌网站模板，纯 HTML + CSS + JS，无需任何框架。

## 预览

https://h-canopy.github.io/canopy-portfolio/

## 特点

- 暗黑金配色（#0d0c0a 暖黑 + #C5A880 古铜金）
- 双主题切换（暗色 / 亮色）
- Canvas 粒子矩阵背景 + 光标跟随太阳
- 技能轨道旋转标签
- 可拖拽印章
- SVG 神话太阳火焰动画
- 双行 Marquee 滚动跑马灯
- 盟友圈浮动卡片弹出
- 世界卷轴展开交互
- 完全响应式（手机 / 平板 / 桌面）
- 入场动画 + 渐显效果

## 怎么用

1. 下载 `Canopy-portfolio.html`
2. 用 VS Code 打开，全局搜索替换：
   - `CANOPY` → 你的名字
   - 项目经历 → 你自己的项目
   - 竞赛 → 你自己的竞赛
   - 联系方式 → 你的信息
3. 替换头像图片
4. 部署到 GitHub Pages / Vercel / Netlify

## 技术栈

- HTML5
- CSS3（Grid + Flexbox + 响应式）
- 原生 JavaScript（Canvas + DOM + 事件）
- 零依赖，无框架

## 字体

- Syne（标题几何字体）
- Bodoni Moda（引言衬线）
- Plus Jakarta Sans（正文字体）
- 均通过 Google Fonts CDN 加载
